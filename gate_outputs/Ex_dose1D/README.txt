
Cet exemple est une simulation d'une source ponctuelle tirant sur un détecteur d'eau.

Une fois executé il est possible de l'analyser avec un script python : tracé du profil... 



Pour l'executer : 




Gate mac/main.mac



Les données de  sorties vont dans le dossier output/



Pour les analyser lancer jupyter:



jupyter notebook



Dans firefox choisir le dossier analysis/ 

puis ouvrir le fichier dose_depth_profile.ipynb




Utiliser SHIFT+ENTREE pour executer les cellules.

La macro ainsi que le fichier dose_depth_profile.ipynb sont commentés pour plus de clarté.
